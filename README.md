# 🚀 PrepForge – Interview Preparation Platform

PrepForge is a full-stack interview preparation platform designed for students and freshers to practice coding, prepare for interviews, manage notes, and track their learning progress.

This platform helps users improve their technical interview preparation through structured resources, coding practice, notes management, and performance tracking.

---

## 📌 Features

### 👨‍💻 Authentication
- User Signup & Login
- JWT Authentication
- Secure Protected Routes

### 📚 Notes Section
- Upload and manage interview notes
- Filter notes by category and difficulty
- Search functionality

### 💻 Coding Preparation
- DSA practice resources
- Coding interview preparation content
- Structured learning sections

### 📊 Dashboard
- Personalized dashboard
- User progress overview
- Quick navigation cards

### 🎨 Modern UI
- Responsive design
- Clean and professional interface
- Mobile-friendly layout
